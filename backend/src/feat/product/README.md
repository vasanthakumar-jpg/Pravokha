# Feature: Product

This module manages the core product catalog and dealer listings.

## Components
- `controller.ts`: Handles HTTP requests for CRUD operations on products.
- `service.ts`: Business logic for product management, including permission checks for dealers vs. admins.
- `route.ts`: Defines API endpoints for product operations.

## Responsibilities
- Product creation and management
- Catalog exploration
- Dealer-specific product filtering
- Admin-level product moderation
